    Link to this function edit(webhook, params) View Source 
    edit(String.t(), Enum.t()) :: map()

    Modifies a given webhook.

    Returns a webhook object upon success or a map containing error information.

    Params

    Must be an enumerable with the fields listed below.

    • name - the default name of the webhook
    • avatar - image for the default webhook avatar
    • channel_id - the new channel id to be moved to

    Refer to this for a broader explanation on the fields and their defaults.
